Install tips
-------------
需要memcache支持
需要打开php短标签标记: short_open_tag = On
导入数据库，配置代码，修改config/allconfig.php内baseurl、数据库、邮箱等关键配置
需要给invite文件夹、shoutbox_new.html等需要进行读写操作的文件、文件夹写入权限
